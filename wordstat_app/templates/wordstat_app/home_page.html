Инструкция по использованию

Чтобы начать работу, выполните следующие шаги:

  1. Шаг 1: Выберите файл Excel

    Нажмите на кнопку "Обзор" и выберите файл Excel с данными для обработки.

  2. Шаг 2: Укажите имя листа

    Введите имя листа в вашем файле Excel, который содержит данные для обработки. Имя листа должно совпадать с тем, что указано в файле.

  3. Шаг 3: Укажите годы и месяцы

    Введите год и месяц начала и окончания периода, для которого нужно провести анализ данных.

  4. Шаг 4: Укажите начальную строку и столбцы

    Укажите начальную строку и номера столбцов для обработки данных:

    • Начальная строка: Укажите номер строки, с которой начнется обработка данных. Нумерация начинается с 1, но программа пропускает заголовок.
    • Столбец с запросами: Укажите букву столбца, содержащего запросы для передачи в Яндекс Wordstat.
    • Столбец для результатов: Укажите букву столбца, в который будут записаны результаты поиска.

После заполнения всех полей, нажмите кнопку "Загрузить и обработать", чтобы отправить файл на сервер и получить результаты анализа.
